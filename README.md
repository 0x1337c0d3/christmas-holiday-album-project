# 14-Day Album Project

A comprehensive creative framework for composing a Bowie "Low" / Jarre "Waiting for Cousteau" inspired album in two weeks.

## Project Overview

**Concept:** Create a split album with:
- **Side A:** 5-6 song sketches (~3-4 min each, ~20 min total)
- **Side B:** 3-4 ambient movements (~5-7 min each, ~20 min total)

**Duration:** 14 days  
**Approach:** Disciplined creative sprints with intentional constraints  
**Philosophy:** Capture ideas, not perfection â€” sketches over polish

## Core Principles

### Creative Constraints

1. **Time Limits:** 1 hour per track (Side A), 1-1.5 hrs per movement (Side B)
2. **Instrument Limits:** Max 2 synths + drums per Side A track
3. **Fixed Palette:** Choose sounds once, stick to them
4. **No Revisiting:** Don't return to previous days' work until Lock-In
5. **Sketches, Not Productions:** Capture ideas, resist perfecting

### Why Constraints Work

- Eliminate decision fatigue
- Force momentum over perfection
- Create cohesive sonic identity
- Prevent analysis paralysis
- Honor the Bowie/Eno Berlin Trilogy aesthetic

## Philosophy

This project embraces the aesthetic of:

- **David Bowie's Berlin Trilogy** (Low, Heroes, Lodger)
- **Brian Eno's ambient works** (Music for Airports, Apollo, Thursday Afternoon)
- **Jean Michel Jarre's long-form compositions** (Waiting for Cousteau, OxygÃ¨ne)

**Key insight:** Some of the best albums were made quickly, with constraints, embracing imperfection. "Low" was recorded in weeks. Eno's ambient works often celebrate the unfinished quality.

**Your sketches have value as sketches.** They don't need to be "finished" to be artistically complete.

## After Day 14

### Option 1: It's Done
- Treat the rough sketches as the final album
- Embrace the unpolished Bowie/Eno aesthetic
- Archive and move to next project

### Option 2: Future Polish
- Take 2-4 weeks away (minimum)
- Return with fresh ears
- Mixing pass (optional)
- Mastering (optional)

### Option 3: Dual Release
- Keep sketches as "demo" version
- Create polished version later
- Both have artistic merit

**There is no wrong answer.**
